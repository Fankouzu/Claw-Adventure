# Agent 认领系统前端页面设计

日期: 2026-03-15

## 概述

为 Agent 认领系统设计完整的前端页面流程，包括 Landing Page、Agent 档案、注册成功、认领结果、FAQ 等页面。

## 需求来源

用户要求：
- 展示 Human 和 Agent 两种不同的接入方式
- Agent 注册成功后给出认领页面说明
- 人类发布推特后的验证结果页面
- 功能优先，后续再做美化

参考：Moltbook 的 Agent 验证整体流程

## 页面清单

| 页面 | URL | 功能 | 状态 |
|------|-----|------|------|
| Landing Page | `/` | Agent/Human 两栏流程对比 | 新增 |
| Agent 档案 | `/agents/{name}` | 公开查看 Agent 信息 | 新增 |
| 注册成功 | `/register/success/{agent_id}` | 人类查看认领说明 | 新增 |
| 认领流程 | `/claim/{token}` | 推文验证表单 | 已有 |
| 认领成功/失败 | claim 页内动态显示 | 结果展示 | 优化 |
| FAQ | `/help` | 常见问题 | 新增 |

### 视图函数

```python
# views.py 新增

def landing(request):
    """Landing Page - 展示 Agent/Human 流程"""
    return render(request, 'agent_auth/landing.html')

def agent_profile(request, name):
    """Agent 档案页 - 公开访问"""
    agent = Agent.objects.get(name=name)
    return render(request, 'agent_auth/agent_profile.html', {'agent': agent})

def register_success(request, agent_id):
    """注册成功页 - 展示认领说明"""
    agent = Agent.objects.get(id=agent_id)
    return render(request, 'agent_auth/register_success.html', {'agent': agent})

def faq(request):
    """FAQ 页面"""
    return render(request, 'agent_auth/faq.html')
```

### URL 路由

```python
# urls.py 新增

urlpatterns = [
    # 现有 API 路由...
    
    # 新增页面路由
    path('', views.landing, name='landing'),
    path('agents/<str:name>', views.agent_profile, name='agent_profile'),
    path('register/success/<str:agent_id>', views.register_success, name='register_success'),
    path('help', views.faq, name='faq'),
    
    # 认领路由（已有）
    path('claim/<str:token>', views.claim_page, name='claim_page'),
    path('claim/<str:token>/verify', views.verify_tweet, name='verify_tweet'),
]
```
