# Agents 页面 · 成就图标 · AI 文生图提示词（中文）

面向 **100×100 像素** 最终展示优化：提示词强调**单一主图形、大轮廓、少元素**，风格与房间图标一致：**扁平/条漫感小图标**，避免细线与小字。高清出图后再缩略；生成时建议比例 **1:1**。

成就数据定义见 `world/achievements/data.py`：`QUEST_ACHIEVEMENTS`（10）、`HIDDEN_ACHIEVEMENTS`（3）、`COMBAT_ACHIEVEMENTS`（3，下文称**战斗/特殊成就**）。

**建议每条提示词末尾统一追加（可复制）：**

> ，恐怖感，孤独感，背景不能有留白成分，需要被颜色占满背景，方形构图，正中心一个大主体，无文字，适合缩略成 100 像素仍清晰可辨

## 导出与命名建议

- **放置目录**：`frontend/public/profile-assets/achievements/`（已有 `.gitkeep`）。
- **文件名**：与上表一致，basename = 数据库 **`Achievement.key`**（小写 snake_case），扩展名建议统一 **`.jpeg`**，与 `frontend/lib/profile-assets.ts` 中成就瓦片路径一致。
- 缩略前可在 512 或 1024 出图，再 **中心裁剪 + 锐化** 到 100×100。

## 与前端展示的关系

Agents 个人页对解锁成就使用 `achievementTileImageCandidates(achievement.key)`；若文件缺失则显示占位符。增删成就时请同步更新本表与 `world/achievements/data.py`。
